# django-blog

## Getting Started

### requirements
- python3
- djanto
  - `pip3 install django`

### サーバー起動
1. `manage.py`のあるところまで移動
2. `python manage.py runserver`

### アクセス先
- [記事一覧箇所](http://localhost:8000/)
- [管理者サイト（記事投稿）](http://localhost:8000/admin/)

## 訂正箇所
- p46, INSTALLED_APPS
  - blog.apps.BlogConfig
